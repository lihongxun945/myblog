# 言川的博客

从2018/07/01以后就在github上用issue写博客了。Issue写博客的好处是：

1. 完美支持 Markdown
2. 可以在线编辑，预览
3. 可以评论
4. 可以打标签
5. 可以写模板

缺点是：

1. 无法管理版本
2. 插入图片不方便
3. 没法自定义主题
